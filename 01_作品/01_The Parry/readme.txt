◆作品について◆

◇使用言語 : C++
◇SDK      : DirectX9
◇開発期間 : 2025/06～2025/09(3か月)
◇ジャンル : 3Dアクション・ボスバトル

◆動作確認済み環境◆
Windows11
（DirectxInput対応のゲームパッド(ジョイスティック)に対応しています。）

◆操作方法◆
◆フルスクリーン◆   		[ F11 ]
◆終了◆   					[ ESC ]

//==============================================
// キーボード
//==============================================
◆移動◆   					[ WASD ]
◆ダッシュ◆   				[ LSHIFT + WASD ]
◆視点移動◆    			[ マウス ]
◆パリィ構え◆  			[ マウス左クリック ]
◆反撃(パリィ成功後)◆  	[ マウス右クリック ]
◆回避◆  					[ SPACE ]
◆絶対反撃(ゲージ消費)◆  	[ Q ]
◆R◆  						[ R ]
◆ポーズ◆ 					[ P ]

//==============================================
// ゲームパッド
//==============================================
◆移動◆   					[ Lスティック ]
◆ダッシュ◆   				[ LB ]
◆視点移動◆    			[ Rスティック ]
◆パリィ構え◆  			[ LT ]
◆反撃(パリィ成功後)◆  	[ RT ]
◆回避◆  					[ B ]
◆絶対反撃(ゲージ消費)◆  	[ L3 ]
◆R◆  						[ R3 ]
◆ポーズ◆ 					[ START ]
